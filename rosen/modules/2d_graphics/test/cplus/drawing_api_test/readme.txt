
pycharm+python环境搭建：https://blog.csdn.net/m0_68111267/article/details/134586653

使用说明
   1）将drawing_demo放在drawing_api_test/demo/drawing目录下。drawing_demo代码见：https://gitee.com/xieyijun3/graphic_graphic_2d/pulls/13
   2）将drawing_api_test文件夹拷贝到本地目录下，用pycharm打开
   3）将要测试的用例名称填写入drawing_api_test/oh_drawing_api.xlsx，并填入yes
   4）drawing_api_test/main.py文件最下面run_all_test函数可以指定性能测试的次数等信息
   5）使用pycharm启动main.py，在drawing_api_test/report中查看结果
